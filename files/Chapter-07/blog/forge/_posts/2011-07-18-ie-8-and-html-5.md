<blockquote><p>"The reason it looked wrong on IE8 (and below) is because I used some of the new HTML5 elements, and apparently older browsers don't understand the nesting structure of the new elements. So when I put, say, an HTML5<br />
<nav> element, then put a list of links inside it, it doesn't pick up on the css identifier "nav ul" like newer browsers. It only recognizes the ul. So, I used the special "if" clause for IE8 and below and created another CSS file with some hacks to get around the issue."<&#47;blockquote><br />
